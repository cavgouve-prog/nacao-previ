Guia
Como Usar
Passo a passo para tirar o máximo do app Nação Previ.
🏛️ Ferramenta de Apoio à Organização de Demandas
1
Cadastre o cliente
Vá em Escritório → Clientes → Novo Cliente. Preencha nome, CPF, WhatsApp e os dados que tiver. O cliente é a base de tudo.
2
Abra o processo
Dentro do cliente, clique em + Processo. Defina o tipo de benefício, a via (INSS ou judicial), a fase atual e o prazo. Você pode editar à medida que o caso evolui.
3
Marque os documentos recebidos
Dentro do processo, há um checklist com os principais documentos previdenciários. Clique no círculo para marcar o que o cliente já entregou.
4
Registre os atendimentos
Cada vez que o cliente ligar, vir ao escritório ou mandar mensagem, registre um atendimento. Isso cria o histórico completo do relacionamento.
5
Lance os andamentos processuais
Sempre que houver uma movimentação no processo (despacho, intimação, sentença, decisão INSS), registre um andamento com a data e o resumo.
6
Configure o financeiro
Registre os honorários combinados e o que já foi recebido. O painel mostra automaticamente o total a receber consolidado de todos os processos.
7
Gere procuração e contrato
Dentro do processo, use o botão Gerar Docs para criar a procuração e o contrato de honorários já preenchidos com os dados do cliente. Copie e cole no Word para ajustes finais.
8
Monitore os prazos
A aba Prazos mostra todos os processos com data limite, organizados por urgência. O Painel alerta automaticamente ao abrir o app quando há prazos vencendo.

💾 Sobre seus dados
Todos os dados são salvos localmente no seu navegador, neste dispositivo.
Ninguém mais tem acesso. Não há servidor. Não há nuvem.
⚠️ Atenção: se limpar o navegador ou trocar de dispositivo, os dados não acompanham.
Use sempre o mesmo computador e navegador para acessar o app.
⚠️ Ferramenta Experimental — Leia com Atenção
Esta é uma versão experimental do sistema de gestão do Nação Previ.
Por ser um sistema em desenvolvimento, podem ocorrer instabilidades, bugs ou perda de dados.

Para minimizar o risco de perda:
Faça backup dos seus dados regularmente usando o botão "Fazer Backup" disponível no Painel.
O arquivo é salvo automaticamente na pasta Downloads do seu computador com a data no nome — exemplo: backup-nacao-previ-11-06-2026.json.
Cada backup gera um arquivo novo com a data do dia. O backup anterior não é apagado.
Recomendamos salvar o arquivo de backup também no Google Drive para maior segurança.
Em caso de perda, use "Restaurar Backup" no Painel, selecione o arquivo e tudo volta automaticamente.

O Nação Previ não se responsabiliza por perda de dados armazenados localmente no navegador.
